If there is no video on live view page, and your browser is IE or IE based browser, please check the below 3 points firstly:

1.   Make sure your OS is Windows(XP, Vista,Win7) and your IE browser or IE based is IE 11.

2.   Make sure your site 192.168.103.49 has been added to Trusted Site and the Security Level is Low.
These settings can be found in IE's Menu: Tool -> Internet Option -> Security.

3.   Make sure ffdshow has been installed on your PC, which can be get from http://sourceforge.net/projects/ffdshow-tryout/
Configure its video decoder enable H.264 and MJPEG use libavcodec library under
Windows Menu: Start -> All Programs -> FFdshow -> Video Decoder Configuration -> Codec
4.   If you not install activeX cab please download it from "http://yourtargetip/activeX/GOKEWeb.cab" and install it.run install.bat in cmd.
5.   Please install 'ffdshow v1.3.4533 [2014-09-29]','microsoft .net framework 4.0'.
osd page : http://192.168.103.47/cgi-bin/demo.cgi?page=3&stream=0
liveview page : http://192.168.103.47/cgi-bin/demo.cgi?page=0&stream=0
enc page : http://192.168.103.47/cgi-bin/demo.cgi?page=1&stream=0
          


If there is still no video on live view page, please download GOKEWeb.cab and follow the below steps:

1.   Extract GOKEWeb.cab to your local directory.

2.   Run install.bat.
You can just double click the file, but it is recommended to open your Windows Command Prompt , cd to the directory and excute 'install.bat'.
If you are on Vista or Win7 OS, please notice you should Run as Administrator : right click the Command Prompt and choose from popped menu.

3.   When you receive the message that the DllRegisterServer succeeded, click OK.
There will be 2 plug-ins need to be registered: GOKERTSPClient.ax,GOKEIPCmrWebPlugIn.ocx. Setup succeed if all of the 2 are registered successfully.


4.   Run vcredist_x86.exe which is Microsoft Visual C++ 2005 SP1 Redistributable Package (x86) , and can be get from http://www.microsoft.com/downloads/details.aspx?FamilyID= 200b2fd9-ae1a-4a14-984d-389c36f85647&DisplayLang=en .
